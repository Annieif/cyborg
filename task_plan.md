# 任务计划：我的世界AI赛博人

## 目标
构建一个生产级质量的 Minecraft AI 赛博人工具，核心基于 Mineflayer，接入外部 AI API，实现与玩家聊天和世界交互。

## 技术栈
- **后端**: Node.js + TypeScript + Mineflayer
- **AI**: OpenAI API / Claude API 兼容接口
- **前端**: React + SVG 矢量图
- **通信**: WebSocket (Socket.io)
- **配置**: YAML/JSON 配置文件
